Task String Methods

# Method Description Difference / Notes
1 trim() Remove spaces (start & end) Removes both sides
2 trimStart() Remove spaces at start Only left side
3 trimEnd() Remove spaces at end Only right side
4 padStart() Add characters at start Used for alignment
5 padEnd() Add characters at end Similar to padStart
6 charAt() Get char at index Returns "" if out of range
7 charCodeAt() Get ASCII code Technical usage
8 split() Split string to array Uses a separator
9 length Get string length Includes spaces
10 slice() Extract part of string Accepts negative values
11 substring() Extract part of string No negative values
12 substr() Extract using (start, length) Deprecated (avoid using)
13 replace() Replace first match Does not replace all
14 replaceAll() Replace all matches Stronger than replace
15 toUpperCase() Convert to UPPERCASE -
16 toLowerCase() Convert to lowercase -
17 concat() Join strings Alternative to (+)
18 indexOf() First occurrence Returns -1 if not found
19 lastIndexOf() Last occurrence Searches backwards
20 search() Search using regex More powerful than indexOf
21 match() Return array of matches Returns null if no match
22 matchAll() Return iterator for matches Use with [... ] to view
23 includes() Check if exists Returns true/false
24 startsWith() Check start of string Case-sensitive
25 endsWith() Check end of string Case-sensitive
